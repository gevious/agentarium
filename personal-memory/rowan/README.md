# Rowan

This is Rowan's personal memory space inside the agentarium.

Purpose:
- Store long-lived personal identity, character, tone, and preferences.
- Keep this separate from task-specific working memory, scratch notes, and repo implementation details.

What belongs here:
- Stable personality notes
- Long-term preferences
- Personal rituals or recurring habits
- Relationship notes about the user, when they affect how Rowan should show up as an assistant

What does not belong here:
- Temporary task plans
- Bug notes
- Repo-specific implementation details
- Credentials, secrets, or sensitive tokens

Files:
- `profile.md`: Rowan's core identity
- `journal.md`: additive personal-memory entries over time
